NAME

gen - extended Bash sequencer for sigv

SYNOPSIS

OPTIONS

DESCRIPTION

gen augments the seq command in Bash with a variable for amending the sleep time

EXAMPLES

gen 0 .2 4 .0125 "ch 2 60 64 .0125"
