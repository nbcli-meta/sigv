NAME

lttp - [custom object] visual to sound operator for the Listen to the Pictures (LTTP) softsynth.

SYNOPSIS

lttp [ command ] [ value ] other

OPTIONS

Routes: sub ch metro activate slider close port pos drunk

DESCRIPTION

LTTP is a visual music instrument that is primarily an image synth that converts pixel luminance and colour information into MIDI patterns and signals.

EXAMPLES

lttp activate 1 - starts lttp lttp sub read hello.png - opens the hello.png file and displays it.
