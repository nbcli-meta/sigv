NAME

aslc - sigv.js command for custom messaging to Crow

SYNOPSIS

aslc [ custom msg ] [ value ]

OPTIONS

route0 - o v i ar adsr var func n2v

DESCRIPTION

aslc is a custom command for communicating with Crow, using Max 4 Live components by Monome.
