NAME

crow - experimental command for controlling Crow sound computer

SYNOPSIS

crow [ lua code ]

OPTIONS

DESCRIPTION

The Crow sound computer interprets Lua code, and the sigv crow command is a prototype for writing and automatically uploading Lua code via USB to Crow module.
