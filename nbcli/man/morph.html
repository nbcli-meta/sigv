NAME

morph - sigv/bash command for carrot (^) transition or transform

SYNOPSIS

morph [ object ] [ iterations ] [ sleep ]

OPTIONS

DESCRIPTION

The morph command currently targets (^) equipped objects, such as geo, proc, and lz, and interpolates randomized value ranges (normalized floats). Related commands include morph0 and morpha.

EXAMPLES

morph geo 250 1 (transforms geo with 1 second intervals)

NOTES

morph is dependent on the rnr command for generating normalized values to transition to.
