NAME

aio - sigv.js audio interface module

SYNOPSIS

aio [ activate ] [ sigx > div ] [ sigy > div ] [ sig > div ] [ phase > div ] [ peak > div ]

OPTIONS

Each of the module parameters: sigx, sigy, sig, phase, and peak have a div subparameter acting as a multiplier for expanding or compressing their entered values.

DESCRIPTION

The aio module activates the DSP in the Max environment, a requirment for analysis of incoming and outgoing audio. Required for aio parameter of sigv.js primitives, and for modules such as grain (granular synth) and lttp (image synth).

EXAMPLES

aio activate 2 (activates audio DSP for incoming audio) aio peak 2 (transfers audio data to bfg module origin values, see man bfg for more information)
