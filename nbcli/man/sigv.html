NAME

sigv - main function for sigv mini-language and CLI utility.

SYNOPSIS

sigv [ flag ] [ *file ]

OPTIONS

-b = jump to sigv project directory and list contents (command list)

-m = create new man page in project directory

-j = launch sigv.js

-- config = open/edit sigv configuration (this file)

-- examples = list of sample instructions

DESCRIPTION

Main sigv mini-language command for initiating sigv environments (i.e. bash/sigv and sigv.js). Mainly used to author the system for performance, experiemnt, and session specifications.

NOTES
